# npm-learn

## 🍊 npm
```bash
npm login

npm publish --access=public

# Seconds time
npm publish
```

## 🐙 GitHub Packages
```bash
# Logging in to GitHub
npm login --registry=https://npm.pkg.github.com --scope=@USER
> Username: USERNAME
> Password: TOKEN
> Email: PUBLIC-EMAIL-ADDRESS

# First time publishing
npm publish --access=public

# Updating the package
npm publish
```
